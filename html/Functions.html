Function Visibility Specifiers

Reference: 

function myFunction() <visibility specifier> returns (bool) {
    return true;
}

Visibility Specifiers - public, private, external, internal.

Example: 

function myFunction() public/private/external/internal returns (bool) {
    return true;
}

Modifiers

Reference: 

function myFunction() external <modifier> returns (bool) {
    return true;
}

Modifiers - pure, view, payable, constant, immutable, anonymous, indexed, virtual, override.

Example: 

function myFunction() external view returns (bool) {
    return true;
}
